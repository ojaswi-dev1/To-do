# **App Name**: TaskTrack

## Core Features:

- Task Creation: Allow users to add new tasks with descriptions and due dates.
- Task Listing: Display tasks in a clear, sortable list, with filtering options.
- Task Editing: Enable users to modify task details.
- Task Completion: Mark tasks as complete with a visual indicator and the ability to archive them.
- Theme Toggle: Enable users to switch between a light and dark theme.

## Style Guidelines:

- Primary color (light theme): Deep Indigo (#3F51B5) for focus and organization.
- Background color (light theme): Very light indigo (#E8EAF6) to keep focus on content.
- Accent color (light theme): Blue (#303F9F) to add a subtle contrast to key elements and buttons.
- Primary color (dark theme): Dark Gray (#212121) for a comfortable viewing experience.
- Background color (dark theme): Black (#000000) to maximize contrast.
- Accent color (dark theme): Teal (#009688) to highlight key elements and improve visibility.
- Body and headline font: 'PT Sans', a modern and readable sans-serif font suitable for both headlines and body text.
- Use clear and consistent icons to represent task status and actions.
- Clean and minimalist layout to ensure clarity and ease of use.
- Subtle transitions for task completion and updates. Smooth transitions when toggling between themes.
